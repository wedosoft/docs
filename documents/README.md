## ⚠️ 작업 시 주의사항

### ✅ **문서 생성 규칙**
1. **기존 문서 먼저 확인** - 비슷한 내용이 있으면 통합
2. **명확한 파일명** - 내용을 즉시 알 수 있는 이름
3. **적절한 분류** - 위의 분류 체계에 맞춰 생성
4. **README 업데이트** - 새 문서 추가 시 이 README 갱신

### ❌ **하지 말아야 할 것**
1. 중복 문서 생성 (예: "완료", "최종", "FINAL" 등 버전 문서)
2. 임시 문서를 그대로 두기
3. 파편화된 작은 문서들 남발
4. 명확하지 않은 파일명 사용

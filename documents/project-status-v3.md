## 🎯 프로젝트 개요

### 목표
CSV 데이터를 기반으로 Freshservice 기술문서를 Docusaurus 기반 웹사이트로 **수작업 변환** (품질 우선)

### 범위
- **대상**: Freshservice 전체 문서 (총 **1,557개 문서**)
- **결과물**: docs.wedosoft.net 완전 한국어 문서 포털
- **언어**: 한국어 (수작업 변환 + 품질 보장)
- **방식**: 카테고리별 순차 처리 (혼재 작업 금지)

### � **수작업 변환 방식** (확정)
```
CSV 원본 → HTML → Markdown → 한국어 리라이팅 → MDX 호환성 처리
```

**수작업 선택 이유:**
- 높은 품질 보장 (기술 문서의 정확성 중요)
- 한국 실무 환경에 맞는 자연스러운 번역
- MDX 호환성 및 Docusaurus 구조 최적화

### 📋 **2. 간단한 작업 진행 방법**
```
1. cd categories/ 
2. 원하는 카테고리 CSV 파일 열기
3. 문서 하나 선택 → description 필드(3번째 컬럼)에서 HTML 추출
4. HTML → Markdown → 한국어 리라이팅
5. 완료된 문서는 별도 표시 또는 이동
6. 다음 문서 반복
```

### 🔧 **3. 진행현황 확인**
```bash
# 카테고리별 파일 확인
ls -la categories/*.csv

# 특정 카테고리 문서 수 확인  
wc -l categories/Freshservice_FAQs.csv

# 작업할 문서 미리보기
head -5 categories/Freshservice_FAQs.csv
```

## 🛠️ 작업 방법론

### 📝 문서 작성 표준
1. **마크다운 리라이팅 지침** 준수
2. **JSX 호환성** 확보 (style 속성 변환)
3. **한국어 번역** 및 실무 중심 리라이팅
4. **이미지 경로** 절대 경로 유지
5. **관련 문서 링크** 최종 단계에서 추가

### ⚡ 작업 효율화 전략
1. **카테고리별 일괄 작업**: 연관 문서를 함께 처리
2. **템플릿 재사용**: 유사한 구조의 문서는 템플릿화
3. **자동화 스크립트**: 반복 작업은 스크립트로 처리
4. **진행상황 추적**: 실시간 진행률 업데이트

# 공통 응답 템플릿으로 미리 작성된 답변 만들기

미리 작성된 답변(Canned Responses)을 활용하여 일관되고 효율적인 고객 대응을 제공할 수 있습니다.

## 미리 작성된 답변의 필요성

### 업무 효율성 향상
- **시간 절약**: 반복적인 답변을 빠르게 작성
- **일관성 유지**: 표준화된 답변으로 서비스 품질 보장
- **오타 방지**: 사전에 검증된 내용으로 실수 최소화

### 고객 만족도 개선
- **빠른 응답**: 신속한 초기 응답으로 고객 만족도 향상
- **전문적 대응**: 체계적이고 전문적인 답변 제공
- **정확한 정보**: 검증된 정보로 신뢰성 있는 서비스

## 미리 작성된 답변 생성

### 기본 생성 방법
1. **관리자 → 고객 서비스 → 미리 작성된 답변**으로 이동
2. **새 답변 추가**를 클릭
3. 답변 제목과 내용을 입력
4. 적용 범위와 권한을 설정

### 답변 구성 요소
- **제목**: 답변을 식별할 수 있는 명확한 제목
- **내용**: 실제 고객에게 발송될 답변 내용
- **태그**: 검색과 분류를 위한 키워드
- **카테고리**: 답변의 주제별 분류

## 효과적인 답변 작성

### 구조화된 답변 템플릿

#### 인사말
```
안녕하세요 {{requester.name}}님,

{{company.name}} 고객지원팀입니다.
문의해 주신 사항에 대해 답변드리겠습니다.
```

#### 본문 내용
```
귀하께서 문의하신 [문제 상황]에 대해 다음과 같이 안내드립니다:

1. [첫 번째 해결 단계]
2. [두 번째 해결 단계]
3. [세 번째 해결 단계]

위 방법으로 해결되지 않을 경우, 추가 지원이 필요하시면 언제든지 연락해 주세요.
```

#### 마무리
```
추가 문의사항이 있으시면 언제든지 연락 주시기 바랍니다.
감사합니다.

{{agent.name}}
{{company.name}} 고객지원팀
```

### 다양한 상황별 템플릿

#### 초기 응답 템플릿
- **접수 확인**: 티켓 접수 확인 및 처리 예정 시간 안내
- **정보 요청**: 추가 정보가 필요한 경우 안내
- **임시 해결책**: 즉시 시도할 수 있는 임시 방법 제시

#### 진행 상황 업데이트
- **조사 중**: 문제 분석 진행 상황 안내
- **외부 확인**: 타 부서나 외부 업체 확인 중인 경우
- **해결 진행**: 해결 작업 진행 상황 공유

#### 해결 완료 응답
- **해결 확인**: 문제 해결 완료 안내
- **만족도 조사**: 서비스 만족도 조사 요청
- **예방 안내**: 재발 방지를 위한 권고 사항

## 동적 변수 활용

### 기본 변수
- `{{requester.name}}`: 요청자 이름
- `{{requester.email}}`: 요청자 이메일
- `{{ticket.id}}`: 티켓 번호
- `{{ticket.subject}}`: 티켓 제목
- `{{agent.name}}`: 담당 상담원 이름
- `{{company.name}}`: 회사 이름

### 고급 변수
- `{{ticket.created_at}}`: 티켓 생성일
- `{{ticket.priority}}`: 티켓 우선순위
- `{{ticket.category}}`: 티켓 카테고리
- `{{requester.department}}`: 요청자 부서

### 조건부 내용
```
{% if ticket.priority == "High" %}
긴급 티켓으로 분류되어 우선 처리하겠습니다.
{% else %}
일반 처리 절차에 따라 진행하겠습니다.
{% endif %}
```

## 팀별 답변 관리

### 공통 답변
- **전사 공통**: 모든 팀이 사용하는 표준 답변
- **정책 안내**: 회사 정책이나 규정 관련 답변
- **일반 절차**: 공통적인 처리 절차 안내

### 팀 전용 답변
- **기술 지원**: IT 기술 관련 전문 답변
- **인사 문의**: 인사 업무 관련 답변
- **재무 문의**: 회계나 결제 관련 답변

### 개인 답변
- **개인 템플릿**: 개별 상담원이 자주 사용하는 답변
- **전문 분야**: 특정 전문 영역의 상세 답변
- **개인 스타일**: 개인의 커뮤니케이션 스타일 반영

## 답변 검색 및 활용

### 효율적인 검색
- **키워드 검색**: 제목이나 내용의 키워드로 검색
- **태그 필터링**: 설정된 태그로 빠른 필터링
- **카테고리 분류**: 주제별로 체계적 분류

### 빠른 적용
- **단축키 활용**: 자주 사용하는 답변의 단축키 설정
- **자동 제안**: 티켓 내용 분석을 통한 자동 추천
- **즐겨찾기**: 개인별 즐겨 사용하는 답변 북마크

## 품질 관리

### 답변 검토
- **정기 검토**: 답변 내용의 정확성과 적절성 검토
- **업데이트**: 정책 변경이나 새로운 정보 반영
- **사용 통계**: 답변별 사용 빈도와 효과 분석

### 피드백 수집
- **고객 반응**: 답변에 대한 고객 만족도 수집
- **상담원 의견**: 실제 사용하는 상담원들의 개선 의견
- **관리자 검토**: 서비스 품질 관점에서의 검토

## 모범 사례

### 작성 가이드라인
- **명확한 표현**: 이해하기 쉬운 간단명료한 문장
- **친근한 톤**: 고객 친화적이면서 전문적인 어조
- **단계별 안내**: 복잡한 내용은 단계별로 구분

### 관리 원칙
- **정기 업데이트**: 변경사항을 즉시 반영
- **버전 관리**: 중요한 변경사항의 이력 관리
- **접근 권한**: 적절한 사용자에게만 편집 권한 부여

### 효과 측정
- **응답 시간**: 미리 작성된 답변 사용 시 응답 시간 단축 효과
- **일관성 지표**: 팀 간 답변 품질의 일관성 정도
- **만족도 개선**: 표준화된 답변이 고객 만족도에 미치는 영향

:::tip 효과적인 템플릿 활용
좋은 미리 작성된 답변은 개인화된 느낌을 주면서도 효율성을 제공합니다. 동적 변수를 적극 활용하여 각 고객에게 맞춤화된 답변을 제공하세요.
:::

:::warning 과도한 의존 주의
미리 작성된 답변에만 의존하지 말고, 각 상황에 맞게 적절히 수정하여 사용하세요. 고객의 특별한 요구사항을 놓치지 않도록 주의해야 합니다.
:::

### 2단계: 새 커스텀 객체 생성

2. &nbsp;새 커스텀 객체를 클릭하세요

## 객체 필드와 객체 레코드 추가

커스텀 객체를 데이터를 캡처하는 테이블로 생각해보세요. 그렇다면,

객체 필드는 정보를 캡처하기 위한 테이블의 카테고리 또는 열입니다.&nbsp;

객체 레코드&nbsp;는 행 또는 필드의 수평적 그룹입니다. 객체 필드의 정보는 객체 레코드마다 고유합니다.&nbsp;

### 객체 필드 추가 방법

To add Object Fields, &lt; "542">


Under the&nbsp;Object Fields Tab,&nbsp;click on&nbsp;Add Object Field. &lt; "50002992904">


Enter the name of the Object Field.
You can choose the object field's type to be one of the following:


Text
Para
Number
Lookup - Currently accepted sources for lookups: Assets, Agents, Departments, Groups, Locations, Requesters, Service Items
Dropdown - Create drop-down fields to reference properties from tickets, problems, changes, release, requesters, and service items.



If you want the entry to this field to be mandatory, mark the&nbsp;Required&nbsp;checkbox.
Similarly, add all the object fields you would require for your custom object.


### 객체 레코드 추가 방법

To add Object Records,&lt;"587">

1. Under the&nbsp;Object Records Tab,&nbsp;click on&nbsp;Add Object Record. &lt; "599">

2. Now add all the relevant data into the object fields to create a record.

3. Click on&nbsp;Save&nbsp;once done.&nbsp;

4. You can now use this custom object in workflows in your automator. You can refer to &lt; "_blank">this article&lt;/a> to understand some detailed use cases to create intelligent automation with custom objects.
